> *You run a sovereignty stack — a Pi, a drive, services, a network. Own the understanding of how it all works, instead of renting it from whoever you'd otherwise have to call for help.*

This wing demystifies the machine from the ground up. The thesis is simple and powerful: **a computer is just a vast pile of on/off switches arranged so that on/off patterns can represent — and transform — anything.** Climb the ladder and that "magic" dissolves into clear, knowable layers.

## The ladder

1. **ELI5** — *How a computer thinks*, with one analogy (light switches that make decisions).
2. **Basic Intro (1–3 hr)** — a guided start: the best video series, the one book to read, and a tour of how the internet moves data.
3. **Deep Dive** — six cited entries, each one layer of the stack:
   - Bits & logic gates — the atoms of computation
   - From gates to the CPU — the fetch–decode–execute cycle
   - Memory & storage — where bits live
   - Operating systems — the manager that makes it usable
   - Networking — IP, TCP & DNS: how a request crosses the world
   - The web & TLS — clients, servers, and encrypted connections
4. **Mastery** — build a CPU in a simulator, trace a real packet, and map your own stack end-to-end.
5. **Resources** — the curated shelf: books, courses, and channels.

## How to read this wing

- Each layer **rests on the one below it**. Bits → gates → CPU → memory → OS → network → web. Read in order the first time; the payoff is seeing the whole tower stand.
- Computing is unusually *knowable* — almost nothing here is mysterious or contested, just abstracted away. Every layer was designed by people and can be understood by people.
- The goal: by the end, you can explain what happens between pressing Enter on a URL and the page appearing — every step.
