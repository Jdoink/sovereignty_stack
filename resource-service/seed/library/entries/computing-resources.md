A shelf that takes you from "what's a bit?" to building a computer — chosen for clarity and longevity. Much of it is **free**.

## Start here (watch / read)

- **Crash Course Computer Science** — ~40 short episodes spanning the whole tower. The best on-ramp.
- **Charles Petzold — *Code*** — builds a computer from a flashlight and a relay, one idea at a time. The first-principles classic; the 2nd edition has a free interactive companion.

## Build it yourself

- **nand2tetris (*The Elements of Computing Systems*)** — construct a full computer from one NAND gate, in a free simulator. The definitive hands-on path.
- **Ben Eater — 8-bit computer** — build a CPU on breadboards with real chips, explained beautifully. For when you want to touch the hardware.

## Go deeper (free, university-grade)

- **Operating Systems: Three Easy Pieces** — the OS textbook, free online. Virtualization, concurrency, persistence.
- **High Performance Browser Networking** (Ilya Grigorik) — free online; the definitive practical guide to TCP, TLS, HTTP, and the browser.

## Reference

- **Cloudflare Learning Center** — clear, accurate explainers for DNS, TLS, and networking concepts.
- **MDN Web Docs** — the canonical reference for HTTP, HTML, CSS, and JavaScript.

## A note on depth

Computing rewards *building* over reading. If you do only one thing from this shelf, do **nand2tetris** — nothing else collapses the "magic" of computers as completely as assembling one yourself from a single gate. Pair it with `traceroute`/`dig` from the **Mastery** tier and you'll understand both halves of your stack: the machine, and the network that connects it.
